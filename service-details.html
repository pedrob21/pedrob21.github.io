Crie seu site apenas inserindo suas informações

Desenvolvido para imobiliárias, mas adaptável para seu negócio.

Utilizando templates em React.js, você insere as informações da sua empresa e o domínio e site são gerados automaticamente de acordo com sua identidade visual e dados, garantindo a melhor experiência para você e seus clientes.

Para este projeto, utilizei Next.js para o middleware, pois a partir de um domínio principal, um site é gerado a partir de seu path. Frontend em React.JS e backend em Python com FastAPI.

Cidades Inteligentes

Desenvolvido para gerenciar frotas e pontos de recarga focados em veículos elétricos.

Com esta aplicação, você pode consultar os dados e estatísticas do seu negócio em tempo real. Ela é focada em bicicletas elétricas e pontos de recarga para carros elétricos, possuindo um banco de dados seguro e robusto, além de proporcionar uma visualização intuitiva das informações.

Utilizando Angular, Python e PostgreSQL, esta aplicação foi desenvolvida como projeto final da disciplina de Banco de Dados, onde o foco era desenvolver um sistema voltado para cidades do futuro e soluções que poderiam ser implementadas. Assim, focamos na redução da emissão de CO2 por meio de dados que ajudam na tomada de decisão.

Simulados de Sistemas Operacionais

Desenvolvido para simular um SO genérico para fins educativos.

Com este site, é possível visualizar como diversas áreas de um SO funcionam, desde processos e funções até paginação e memória.

O projeto doi desenvolvido na disciplina de Sistemas Operacionais, onde utilizamos C, Node.JS, MongoDB e React.JS para desenvolver as simulações dos processos e visualizações intuitivas das ações mais difíceis de compreender.
